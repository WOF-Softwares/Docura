The Tech Stack

Built with modern, performant technologies for the best user experience.

Tauri 2.8 (Rust) React 19 Vite 7 Monaco Editor WebKitGTK react-markdown remark-gfm jspdf

Why Tauri + React?

  • • WebKitGTK vs Chromium: Tauri uses system WebKit instead of bundling Chromium. Result? 780 MB vs 1011 MB — 231 MB saved.
  • • Rust Backend: Native performance and security. File I/O and system integration in Rust, not Node.js.
  • • Native Integration: Respects system themes, uses native dialogs, integrates with tiling window managers.

Ready to Get Started?

Or build from source for the latest features: 

git clone https://github.com/WOF-Softwares/Docura.git
cd Docura
npm install
./dev.sh

Requirements: Node.js 18+, Rust (via rustup), WebKitGTK 4.1
