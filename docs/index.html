🎨 Omarchy Integration - Built for DHH's Workflow

🍣

What is Omarchy?

Omarchy is DHH's (creator of Ruby on Rails) curated development environment for Arch Linux. It's opinionated, beautiful, and just works - like Rails for your entire dev setup.

🔍

Auto-Detection

Docura automatically detects Omarchy and offers theme synchronization. Zero configuration needed!

🔄

Real-time Sync

Change your Omarchy theme in the terminal → Docura updates automatically every 30 seconds

🔒

Respects Control

When synced, Omarchy controls themes. Manual overrides are blocked - this is intentional and respectful!

🎨

12 Theme Mappings

Dracula, Catppuccin, Nord, Gruvbox, Tokyo Night, and more - all mapped perfectly

💎 For DHH and the Ruby Community

This integration was built with deep respect for DHH's philosophy. When you enable Omarchy sync, Docura doesn't fight your setup - it embraces it.

Just like Rails respects convention, Docura respects Omarchy. Opinionated, simple, it just works. 🙏

The Tech Stack

Built with modern, performant technologies for the best user experience.

Tauri 2.8 (Rust) React 19 Vite 7 Monaco Editor WebKitGTK react-markdown remark-gfm jspdf

Why Tauri + React?

  • • WebKitGTK vs Chromium: Tauri uses system WebKit instead of bundling Chromium. Result? 780 MB vs 1011 MB — 231 MB saved.
  • • Rust Backend: Native performance and security. File I/O and system integration in Rust, not Node.js.
  • • Native Integration: Respects system themes, uses native dialogs, integrates with tiling window managers.
