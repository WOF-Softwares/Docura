## üìä Technical Examples

### Markdown vs .docx Comparison

**Markdown (5 bytes):**

```markdown
**Bold text**
```

**XML/.docx (200+ bytes):**

```xml
<w:p>
  <w:pPr>
    <w:pStyle w:val="Normal"/>
    <w:rPr>
      <w:b/>
    </w:rPr>
  </w:pPr>
  <w:r>
    <w:rPr>
      <w:b/>
    </w:rPr>
    <w:t>Bold text</w:t>
  </w:r>
</w:p>
```

**10x the code for the same result!**

### JSON + Markdown Example

```json
{
  "title": "My Document",
  "author": "Jane Doe",
  "date": "2025-10-13",
  "tags": ["tech", "ai", "future"],
  "content": "# Introduction\n\nThis is **markdown** inside JSON."
}
```
